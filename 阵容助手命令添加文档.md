## 4️⃣ GameStatus.vue 组件显示方式修改

### 📍 文件位置
`src/components/GameStatus.vue`

### 🔍 查找位置
在文件中搜索 `Unlimitedlineup`，找到阵容助手组件的引用行。

### 📝 需要修改的内容

**修改前（使用 v-if）：**
```vue
<!-- 阵容助手（提取组件） -->
<Unlimitedlineup v-if="activeSection === 'tools'" />
```

**修改后（使用 v-show）：**
```vue
<!-- 阵容助手（提取组件） -->
<Unlimitedlineup v-show="activeSection === 'tools'" />
```

### ⚠️ 重要说明

**为什么要改为 v-show？**

- `v-if` 会完全销毁和重建组件，导致组件状态丢失
- `v-show` 只是切换 CSS `display` 属性，组件始终存在，状态得以保留
- 阵容助手需要保持组件状态，避免频繁重新初始化

### 📌 修改示例

**注意：只有阵容助手（Unlimitedlineup）需要修改，其他组件保持原样！**

```vue
<!-- 其他组件保持原样，不需要修改 -->

<!-- 阵容助手（提取组件）- 只有这个需要改 -->
<Unlimitedlineup v-show="activeSection === 'tools'" />

<!-- 其他组件保持原样 -->
```

## 5️⃣ 命令功能说明

| 命令 | 功能 | 使用场景 |
|------|------|----------|
| `hero_exchange` | 武将交换 | 阵容切换时交换两个武将的位置 |
| `hero_gointobattle` | 武将上阵 | 将武将放到阵容指定位置 |
| `hero_gobackbattle` | 武将下阵 | 将武将移出阵容 |
| `hero_heroupgradelevel` | 武将升级 | 提升武将等级 |
| `hero_heroupgradeorder` | 武将进阶 | 提升武将进阶等级 |
| `hero_rebirth` | 武将重生 | 重置武将属性并返还资源 |
| `artifact_load` | 鱼灵装备 | 给武将装备鱼灵 |
| `artifact_unload` | 鱼灵卸下 | 卸下武将的鱼灵 |
| `lordweapon_changedefaultweapon` | 切换玩具 | 切换默认玩具 |
| `pearl_replaceskill` | 鱼珠技能替换 | 替换鱼珠技能 |
| `pearl_exchangeskill` | 鱼珠技能交换 | 交换两个鱼珠的技能 |
| `pearl_unloadskill` | 鱼珠技能卸下 | 卸下鱼珠技能 |
| `legion_research` | 俱乐部研究 | 进行俱乐部科技研究 |
| `legion_resetresearch` | 重置俱乐部研究 | 重置俱乐部科技研究 |
| `legion_payloadsignup` | 蟠桃报名 | 报名参加蟠桃活动 |
| `car_research` | 车辆研究 | 进行车辆科技研究 |
| `car_claimpartconsumereward` | 车辆零件消耗奖励 | 领取车辆零件消耗奖励 |
| `activity_buystoregoods` | 活动商店购买 | 在活动商店购买物品 |

## 6️⃣ 常见问题

### Q: 如果忘记添加某个命令会怎样？
A: 会出现 `Unknown cmd: xxx` 错误，导致 WebSocket 断开连接。

### Q: 如果响应映射没有添加会怎样？
A: 命令发送后可能无法正确接收响应，导致功能卡住或超时。

### Q: 这些命令会影响其他功能吗？
A: 不会，这些命令只用于阵容助手功能，不会影响其他模块。
