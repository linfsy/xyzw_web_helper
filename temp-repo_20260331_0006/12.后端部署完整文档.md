## 1. 项目概述

### 1.1 项目信息

- **项目名称**: xyzw_server_agent
- **仓库地址**: https://github.com/w1249178256/xyzw_server_agent
- **项目描述**: 基于 Spring Boot 的咸鱼之王游戏服务端代理
- **主要功能**: 
  - 角色配置管理
  - 自动战斗（爬塔、竞技场等）
  - 每日任务自动调度
  - 系统监控
  - 数据缓存

### 1.2 技术栈

| 技术 | 版本 | 说明 |
|------|------|------|
| Java | 17+ | 运行环境 |
| Spring Boot | 3.5.3 | 后端框架 |
| Spring Security | - | 安全框架 |
| MyBatis-Plus | - | ORM框架 |
| MySQL | 8.0+ | 数据库 |
| Redis | 6.0+ | 缓存 |
| WebSocket | - | 游戏通信 |
| JWT | - | 身份认证 |

## 3. 软件安装

### 3.1 安装 Java JDK 17+

#### Windows 系统

**方法一：官网下载安装**

1. 访问 Oracle 官网下载页面：
   ```
   https://www.oracle.com/java/technologies/downloads/
   ```

2. 选择 Windows 版本，下载 `.exe` 安装包

3. 双击安装包，按照提示完成安装

4. 配置环境变量：
   - 右键"此电脑" → "属性" → "高级系统设置" → "环境变量"
   - 新建系统变量：
     ```
     变量名: JAVA_HOME
     变量值: C:\Program Files\Java\jdk-21
     ```
   - 编辑 Path 变量，添加：
     ```
     %JAVA_HOME%\bin
     ```

5. 验证安装：
   ```powershell
   java -version
   ```
   输出示例：
   ```
   java version "21.0.10" 2026-01-20 LTS
   Java(TM) SE Runtime Environment (build 21.0.10+8-LTS-217)
   Java HotSpot(TM) 64-Bit Server VM (build 21.0.10+8-LTS-217, mixed mode, sharing)
   ```

**方法二：使用 Scoop 安装（推荐）**

```powershell
# 安装 Scoop（如果未安装）
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
irm get.scoop.sh | iex

# 安装 Java
scoop install openjdk17
```

#### Linux 系统

```bash
# Ubuntu/Debian
sudo apt update
sudo apt install openjdk-17-jdk -y

# CentOS/RHEL
sudo yum install java-17-openjdk java-17-openjdk-devel -y

# 验证安装
java -version
```

#### macOS 系统

```bash
# 使用 Homebrew 安装
brew install openjdk@17

# 配置环境变量
sudo ln -sfn /opt/homebrew/opt/openjdk@17/libexec/openjdk.jdk /Library/Java/JavaVirtualMachines/openjdk-17.jdk

# 验证安装
java -version
```

## 5. 配置说明

### 5.1 主配置文件 application.yml

创建或修改 `src/main/resources/application.yml`：

```yaml
# 服务器配置
server:
  port: 8080
  servlet:
    context-path: /

# Spring 配置
spring:
  # 数据源配置
  datasource:
    url: jdbc:mysql://localhost:3306/xyzw?useUnicode=true&characterEncoding=utf8&useSSL=false&serverTimezone=Asia/Shanghai
    username: root
    password: your_mysql_password
    driver-class-name: com.mysql.cj.jdbc.Driver
    # 连接池配置
    hikari:
      minimum-idle: 5
      maximum-pool-size: 20
      idle-timeout: 30000
      pool-name: XyzwHikariPool
      max-lifetime: 1800000
      connection-timeout: 30000

  # Redis 配置
  redis:
    host: localhost
    port: 6379
    password:           # 如果设置了密码，填写密码
    database: 0
    timeout: 10000ms
    lettuce:
      pool:
        max-active: 8
        max-wait: -1ms
        max-idle: 8
        min-idle: 0

  # 任务调度配置
  task:
    scheduling:
      pool:
        size: 20
      thread-name-prefix: task-

  # JWT 配置
  jwt:
    secret: your-jwt-secret-key-at-least-256-bits-long
    expiration: 86400000  # 24小时（毫秒）

# MyBatis-Plus 配置
mybatis-plus:
  mapper-locations: classpath:mapper/*.xml
  type-aliases-package: xyzw_helper.entity
  configuration:
    map-underscore-to-camel-case: true
    log-impl: org.apache.ibatis.logging.stdout.StdOutImpl

# 日志配置
logging:
  level:
    root: INFO
    xyzw_helper: DEBUG
  pattern:
    file: "%d{yyyy-MM-dd HH:mm:ss} [%thread] %-5level %logger{50} - %msg%n"
  file:
    name: logs/xyzw_agent.log

# 游戏服务器配置
game:
  server:
    host: game.example.com
    port: 8080
    ws-url: wss://game.example.com/ws
```

### 5.2 开发环境配置 application-dev.yml

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/xyzw?useUnicode=true&characterEncoding=utf8&useSSL=false&serverTimezone=Asia/Shanghai
    username: root
    password: your_dev_password

  redis:
    host: localhost
    port: 6379

logging:
  level:
    xyzw_helper: DEBUG

game:
  debug: true
```

### 5.3 生产环境配置 application-prod.yml

```yaml
spring:
  datasource:
    url: jdbc:mysql://your-prod-server:3306/xyzw?useUnicode=true&characterEncoding=utf8&useSSL=true&serverTimezone=Asia/Shanghai
    username: xyzw_user
    password: ${DB_PASSWORD}  # 从环境变量读取

  redis:
    host: ${REDIS_HOST:localhost}
    port: ${REDIS_PORT:6379}
    password: ${REDIS_PASSWORD:}

logging:
  level:
    root: INFO
    xyzw_helper: INFO

game:
  debug: false
```

### 5.4 配置项说明

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| `server.port` | 服务端口 | 8080 |
| `spring.datasource.url` | 数据库连接地址 | - |
| `spring.datasource.username` | 数据库用户名 | root |
| `spring.datasource.password` | 数据库密码 | - |
| `spring.redis.host` | Redis 主机地址 | localhost |
| `spring.redis.port` | Redis 端口 | 6379 |
| `spring.jwt.secret` | JWT 密钥（至少256位） | - |
| `spring.jwt.expiration` | JWT 过期时间（毫秒） | 86400000 |
| `spring.task.scheduling.pool.size` | 任务线程池大小 | 20 |

## 8. 常见问题

### 8.1 端口被占用

**问题：** 启动时报错 `Port 8080 is already in use`

**解决方案：**

```bash
# Windows - 查找占用端口的进程
netstat -ano | findstr :8080

# 终止进程
taskkill /PID <PID> /F

# 或者修改配置文件中的端口
server:
  port: 8081
```

```bash
# Linux/macOS - 查找占用端口的进程
lsof -i :8080

# 终止进程
kill -9 <PID>
```

### 8.2 数据库连接失败

**问题：** 启动时报错 `Communications link failure`

**解决方案：**

1. 检查 MySQL 服务是否启动：
   ```bash
   # Windows
   net start MySQL80
   
   # Linux
   sudo systemctl start mysql
   ```

2. 检查数据库连接配置是否正确

3. 检查防火墙是否开放 3306 端口

4. 检查用户名密码是否正确

### 8.3 Redis 连接失败

**问题：** 启动时报错 `Unable to connect to Redis`

**解决方案：**

1. 检查 Redis 服务是否启动：
   ```bash
   # Windows
   redis-server.exe
   
   # Linux
   sudo systemctl start redis
   ```

2. 检查 Redis 配置是否正确

3. 如果 Redis 设置了密码，确保配置文件中填写了密码

### 8.4 JWT 密钥长度不足

**问题：** 启动时报错 `The specified key byte array is 256 bits`

**解决方案：**

确保 `spring.jwt.secret` 配置的密钥长度至少为 256 位（32 字节）：

```yaml
spring:
  jwt:
    secret: your-jwt-secret-key-at-least-256-bits-long-please-use-a-long-string
```

### 8.5 内存不足

**问题：** 启动时报错 `Java heap space`

**解决方案：**

增加 JVM 堆内存：

```bash
java -Xms512m -Xmx1024m -jar target/xyzw_helper.jar
```

### 8.6 定时任务不执行

**问题：** 配置了定时任务但不执行

**解决方案：**

1. 检查 Cron 表达式是否正确
2. 检查任务调度是否启用：
   ```yaml
   spring:
     task:
       scheduling:
         enabled: true
   ```
3. 查看日志是否有错误信息

### 9.3 创建 docker-compose.yml

在项目根目录创建 `docker-compose.yml`：

```yaml
version: '3.8'

services:
  app:
    build: .
    container_name: xyzw_agent
    ports:
      - "8080:8080"
    environment:
      - SPRING_PROFILES_ACTIVE=prod
      - DB_HOST=mysql
      - DB_PORT=3306
      - DB_NAME=xyzw
      - DB_USERNAME=root
      - DB_PASSWORD=password
      - REDIS_HOST=redis
      - REDIS_PORT=6379
    depends_on:
      - mysql
      - redis
    networks:
      - xyzw_network

  mysql:
    image: mysql:8.0
    container_name: xyzw_mysql
    environment:
      - MYSQL_ROOT_PASSWORD=password
      - MYSQL_DATABASE=xyzw
      - MYSQL_CHARACTER_SET_SERVER=utf8mb4
      - MYSQL_COLLATION_SERVER=utf8mb4_unicode_ci
    volumes:
      - mysql_data:/var/lib/mysql
      - ./src/main/resources/db/init.sql:/docker-entrypoint-initdb.d/init.sql
    ports:
      - "3306:3306"
    networks:
      - xyzw_network
    restart: unless-stopped

  redis:
    image: redis:6-alpine
    container_name: xyzw_redis
    ports:
      - "6379:6379"
    volumes:
      - redis_data:/data
    networks:
      - xyzw_network
    restart: unless-stopped

networks:
  xyzw_network:
    driver: bridge

volumes:
  mysql_data:
  redis_data:
```
