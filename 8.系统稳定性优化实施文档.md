### 3.1 积攒列表优化

#### 3.1.1 智能任务去重（1.1）

**功能描述**：
同时检查任务名称、类型、账号列表和任务类型，避免重复任务被添加到队列中。

**实现逻辑**：
1. 检查任务名称是否相同
2. 检查运行类型是否相同
3. 检查选中的账号列表是否相同（长度和每个元素）
4. 检查选中的任务类型是否相同（排序后比较）

**涉及文件**：
- `src/stores/batchTaskStore.ts` - `addToTaskQueue` 函数
- `src/views/BatchDailyTasks.vue` - 多处任务添加逻辑

**代码实现**：

**文件1：src/stores/batchTaskStore.ts**
```typescript
function addToTaskQueue(task: {
  id?: number;
  name?: string;
  selectedTokens?: string[];
  selectedTasks?: string[];
  runType?: string;
}) {
  // 同时检查任务名称、运行类型、账号列表和任务类型
  const isTaskExists = taskQueue.value.some(existingTask => {
    // 检查任务名称和类型
    if (existingTask.name === task.name && existingTask.runType === task.runType) {
      // 检查选中的账号是否相同
      const existingTokens = existingTask.selectedTokens || [];
      const newTokens = task.selectedTokens || [];
      const sameTokens = existingTokens.length === newTokens.length && 
             existingTokens.every(tokenId => newTokens.includes(tokenId));
      
      // 检查选中的任务类型是否相同
      const existingTasks = existingTask.selectedTasks || [];
      const newTasks = task.selectedTasks || [];
      const sameTasks = JSON.stringify(existingTasks.sort()) === JSON.stringify(newTasks.sort());
      
      // 只有名称、类型、账号和任务类型都相同才认为是重复任务
      return sameTokens && sameTasks;
    }
    return false;
  });
  
  if (!isTaskExists) {
    taskQueue.value.push(task);
    console.log('[BatchTask] 任务已加入队列:', task.name);
  } else {
    console.log('[BatchTask] 任务已在队列中，跳过重复添加:', task.name);
  }
}
```

**文件1：src/views/BatchDailyTasks.vue（加载队列）**
```javascript
const loadTaskQueueFromStorage = () => {
  // 如果队列已经有数据，先清空（避免热更新时重复添加）
  if (batchTaskStore.taskQueue.length > 0) {
    batchTaskStore.clearTaskQueue();
  }
  
  // 优先从新的键名加载
  const savedQueue = safeLocalStorage.getItem('batch_task_queue');
  if (savedQueue && typeof savedQueue === 'string' && savedQueue !== 'undefined') {
    try {
      const parsedQueue = JSON.parse(savedQueue);
      if (Array.isArray(parsedQueue)) {
        parsedQueue.forEach(task => {
          batchTaskStore.addToTaskQueue(task);
        });
        if (batchTaskStore.taskQueue.length > 0) {
          addLog({
            time: new Date().toLocaleTimeString(),
            message: `=== 从本地存储加载了 ${batchTaskStore.taskQueue.length} 个积攒任务 ===`,
            type: "info",
          });
        }
      }
    } catch (error) {
      console.error('加载任务队列失败:', error);
      // 重要修复：加载失败时不清空队列，保留现有队列数据
      // 避免数据丢失，只在明确需要清空时才清空
      addLog({
        time: new Date().toLocaleTimeString(),
        message: `=== 加载积攒队列失败: ${error.message}，保留现有队列 ===`,
        type: "warning",
      });
    }
  }
  
  // 清理旧的键名
  safeLocalStorage.removeItem('taskQueue');
};
```

**文件1：src/views/BatchDailyTasks.vue（暂停状态检测）**
```javascript
// 暂停时间检测
const isPauseTime = ref({
  paused: false,
  reason: '',
  nextResumeTime: null
});

const updatePauseStatus = () => {
  const now = new Date();
  const currentHour = now.getHours();
  const currentMinute = now.getMinutes();
  const currentDay = now.getDay(); // 0=周日, 6=周六
  
  // 检查抢符时间（20:00-20:30）
  if (currentHour === 20 && currentMinute >= 0 && currentMinute <= 30) {
    isPauseTime.value = {
      paused: true,
      reason: '抢符时间',
      nextResumeTime: new Date(now.getTime() + (30 - currentMinute) * 60 * 1000)
    };
    return;
  }
  
  // 检查周六盐场（周六20:00-21:00）
  if (currentDay === 6 && currentHour === 20) {
    isPauseTime.value = {
      paused: true,
      reason: '周六盐场',
      nextResumeTime: new Date(now.getTime() + (60 - currentMinute) * 60 * 1000)
    };
    return;
  }
  
  // 检查周日盐场（周日20:00-21:00）
  if (currentDay === 0 && currentHour === 20) {
    isPauseTime.value = {
      paused: true,
      reason: '周日盐场',
      nextResumeTime: new Date(now.getTime() + (60 - currentMinute) * 60 * 1000)
    };
    return;
  }
  
  // 检查游戏更新时间（根据配置）
  const updateTime = getGameUpdateTime(); // 从配置中获取
  if (updateTime && isDuringUpdate(now, updateTime)) {
    isPauseTime.value = {
      paused: true,
      reason: '游戏更新',
      nextResumeTime: getUpdateEndTime(updateTime)
    };
    return;
  }
  
  // 不在暂停时间
  isPauseTime.value = {
    paused: false,
    reason: '',
    nextResumeTime: null
  };
};

// 每分钟检查一次暂停状态
setInterval(updatePauseStatus, 60 * 1000);
```

```javascript
export const safeLocalStorage = {
  getItem(key) {
    try {
      return localStorage.getItem(key);
    } catch (error) {
      console.error(`localStorage getItem error for key "${key}":`, error);
      return null;
    }
  },
  
  setItem(key, value) {
    try {
      localStorage.setItem(key, value);
      return true;
    } catch (error) {
      console.error(`localStorage setItem error for key "${key}":`, error);
      
      // 如果是存储空间不足，尝试清理一些数据
      if (error.name === 'QuotaExceededError') {
        console.warn('Storage quota exceeded, attempting to clean up...');
        cleanupStorage();
        
        // 重试一次
        try {
          localStorage.setItem(key, value);
          return true;
        } catch (retryError) {
          console.error('Retry failed:', retryError);
          return false;
        }
      }
      
      return false;
    }
  },
  
  removeItem(key) {
    try {
      localStorage.removeItem(key);
      return true;
    } catch (error) {
      console.error(`localStorage removeItem error for key "${key}":`, error);
      return false;
    }
  },
  
  clear() {
    try {
      localStorage.clear();
      return true;
    } catch (error) {
      console.error('localStorage clear error:', error);
      return false;
    }
  }
};

// 清理存储空间
function cleanupStorage() {
  // 清理过期的日志
  const logKeys = Object.keys(localStorage).filter(key => key.startsWith('logs_'));
  logKeys.forEach(key => localStorage.removeItem(key));
  
  // 清理过期的任务标记
  const taskKeys = Object.keys(localStorage).filter(key => key.startsWith('task_executing_'));
  taskKeys.forEach(key => {
    const executionTime = parseInt(localStorage.getItem(key) || '0');
    if (Date.now() - executionTime > 30 * 60 * 1000) {
      localStorage.removeItem(key);
    }
  });
}
```

```javascript
const saveTaskQueueToStorage = () => {
  try {
    const queue = batchTaskStore.taskQueue || [];
    if (!Array.isArray(queue)) {
      console.error('任务队列不是数组:', queue);
      return;
    }
    
    // 限制队列长度，避免存储过大的数据
    const limitedQueue = queue.slice(0, 50);
    
    // 序列化前进行数据清理，移除可能导致循环引用的属性
    const sanitizedQueue = limitedQueue.map(task => {
      const sanitizedTask = { ...task };
      
      // 移除函数和复杂对象
      Object.keys(sanitizedTask).forEach(key => {
        const value = sanitizedTask[key];
        if (typeof value === 'function') {

        } else if (value instanceof Date) {

        } else if (value instanceof HTMLElement) {
          delete sanitizedTask[key];
        }
      });
      
      // 处理 params 属性，只保留简单数据类型
      if (sanitizedTask.params) {

const sanitizeParams = (params) => {

    return params;

        typeof value === 'number' || 
        typeof value === 'boolean' ||
        value === null ||
        (Array.isArray(value) && value.every(v => typeof v !== 'function')) ||

      sanitized[key] = value;
    }
  });
```

```javascript
for (let batchIndex = 0; batchIndex < totalBatches; batchIndex++) {
```

```javascript
  const startIndex = batchIndex * batchSize;

    addLog({
      time: new Date().toLocaleTimeString(),
      message: `第 ${batchIndex + 1} 批执行失败: ${batchError.message}，继续执行下一批`,
      type: "error",
    });
    console.error(`批次 ${batchIndex + 1} 执行失败:`, batchError);
  }
  
  // 等待下一批
  if (batchIndex < totalBatches - 1 && !batchTaskStore.shouldStop.value) {
    await new Promise(resolve => setTimeout(resolve, batchDelay));
  }
}
```

#### 3.4.4 暂停检测（4.4）

**功能描述**：
批处理等待期间也会检查暂停状态，检测到暂停时，将任务加入积攒队列。

**实现逻辑**：
1. 在批处理等待期间定期检查暂停状态
2. 检测到暂停时，将任务加入积攒队列

**涉及文件**：
- `src/views/BatchDailyTasks.vue` - 批处理等待逻辑

**代码实现**：

**文件：src/views/BatchDailyTasks.vue**
```javascript
// 批处理等待期间检查暂停状态
if (batchIndex < totalBatches - 1 && !batchTaskStore.shouldStop.value) {
  addLog({
    time: new Date().toLocaleTimeString(),
    message: `=== 第 ${batchIndex + 1} 批执行完成，等待 ${batchDelay / 1000} 秒后执行下一批 ===`,
    type: "info",
  });

  let remainingSeconds = batchDelay / 1000;
  
  while (remainingSeconds > 0 && !batchTaskStore.shouldStop.value) {
    // 检查暂停状态
    if (isPauseTime.value.paused) {
      addLog({
        time: new Date().toLocaleTimeString(),
        message: `=== 批量任务被暂停: 当前处于${isPauseTime.value.reason}，已加入积攒队列 ===`,
        type: "info",
      });
      
      // 将剩余任务加入积攒队列
      const remainingTokens = availableTokens.slice((batchIndex + 1) * batchSize);
      batchTaskStore.addToTaskQueue({
        name: task.name,
        selectedTokens: remainingTokens,
        selectedTasks: taskTasksList,
        runType: task.runType,
        id: task.id
      });
      
      selectedTokens.value = [...originalSelectedTokens];
      return;
    }
    
    await new Promise(resolve => setTimeout(resolve, 1000));
    remainingSeconds--;
  }
}
```

**测试方法**：
1. 启动批量任务
2. 在等待期间设置暂停时间
3. 验证任务被暂停并加入积攒队列
4. 等待暂停时间结束
5. 验证任务自动恢复执行

**注意事项**：
